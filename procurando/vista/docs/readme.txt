This scripts are based on megaupload project, owned by Raditha Dissanyake ( http://www.raditha.com )

You have to upload this script to a web server wich can use PHP & Perl scripts.

Make sure you place the CGI script in a folder with correct execution permissions.

You have to modify the temp folder path at upload.cgi & ajax-process.php scripts.
